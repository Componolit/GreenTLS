====== Vetting SSL Usage in Applications with SSL INT ======

- http://www.cs.northwestern.edu/~ychen/Papers/SSLint.pdf

Security vulnerabilites in applications using TLS are often caused by poor API design or inexperienced developers.
SSLINT detects incorrect usage of TLS APIs by statically analysing application code.
